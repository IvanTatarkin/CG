# Лабораторная работа №1: Основы 3D-графики

## Описание

Программа отображает сферу с пульсирующим масштабом. Сфера генерируется параметрическими уравнениями (~100 вершин), использует перспективную проекцию и анимируется по синусоиде (sin(time)). Добавлены элементы UI для поворота камеры.

## Требования

- Vulkan SDK
- GLFW3
- GLM (OpenGL Mathematics)
- ImGUI
- CMake (версия 3.20 или выше)
- Компилятор C++17

## Сборка

1. Установите зависимости:
   ```bash
   # Ubuntu/Debian
   sudo apt-get install libvulkan-dev libglfw3-dev libglm-dev cmake
   
   # Или используйте пакетный менеджер вашего дистрибутива
   ```

2. Клонируйте или добавьте ImGUI как подмодуль:
   ```bash
   git submodule add https://github.com/ocornut/imgui.git third_party/imgui
   ```

3. Скомпилируйте шейдеры:
   ```bash
   ./compile_shaders.sh
   ```

4. Соберите проект:
   ```bash
   mkdir build
   cd build
   cmake ..
   make
   ```

5. Запустите программу:
   ```bash
   ./Lab1_3DGraphics
   ```

## Использование

- Используйте слайдеры в окне "Camera Controls" для поворота камеры (Yaw и Pitch)
- Сфера автоматически пульсирует с масштабом, изменяющимся по синусоиде
- Масштаб отображается в UI

## Структура проекта

- `src/` - исходные файлы
  - `main.cpp` - главный файл приложения
  - `vulkan_renderer.cpp` - рендерер Vulkan
  - `sphere_generator.cpp` - генератор сферы
  - `camera.cpp` - управление камерой
  - `math_utils.cpp` - математические утилиты
- `include/` - заголовочные файлы
- `shaders/` - GLSL шейдеры
  - `vert.glsl` - вершинный шейдер
  - `frag.glsl` - фрагментный шейдер

## Особенности реализации

- Сфера генерируется параметрическими уравнениями с использованием сферических координат
- Перспективная проекция реализована через матрицу проекции
- Анимация пульсации использует sin(time) для изменения масштаба
- UI реализован с помощью ImGUI для управления камерой
